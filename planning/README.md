# planning/ — 기획 문서 폴더

이 폴더에 기획 문서를 넣으면 Claude Code가 자동으로 `SPEC.md`를 생성합니다.

---

## 지원 파일 형식

| 확장자 | 설명 |
|--------|------|
| `.md` | Markdown 문서 |
| `.txt` | 일반 텍스트 |
| `.pdf` | PDF 문서 |

---

## 사용법

### 1. 기획 문서 넣기

이 폴더에 기획 문서를 넣으세요. 파일이 여러 개여도 됩니다.

```
planning/
├── 기획서.md
├── 요구사항.txt
└── 화면설계.pdf
```

### 2. SPEC.md 자동 생성

Claude Code에 요청:

```
planning 폴더를 읽고 SPEC.md를 작성해줘
```

### 3. 검토 및 수정

생성된 `SPEC.md`를 검토하세요. `<!-- TODO: 작성 필요 -->` 주석이 있는 부분은 직접 작성이 필요합니다.

### 4. 프로젝트 생성

```
SPEC.md를 읽고 프로젝트를 생성해줘
```

---

## 기획 문서 작성 팁

더 정확한 SPEC.md를 생성하려면 아래 내용을 포함하세요:

- **프로젝트 설명**: 프로젝트 이름, 목적, 대상 사용자
- **사용자 역할**: 관리자, 일반 사용자 등 역할 구분
- **핵심 기능**: 어떤 데이터를 다루고 어떤 기능이 필요한지
- **화면 구성**: 필요한 페이지와 각 페이지의 구성 요소
- **AI 기능** (선택): 챗봇, 자동 분석 등 AI 관련 요구사항
- **디자인** (선택): 브랜드 컬러, UI 스타일 가이드

---

## 참고

- 이 폴더의 내용은 `.gitignore`에 의해 커밋에서 제외됩니다 (기획 문서가 민감할 수 있으므로)
- 커밋에 포함하려면 `.gitignore`에서 `planning/` 항목을 제거하세요
